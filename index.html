Szkolenie przeznaczone jest dla osób pragnących rozpocząć przygodę z tworzeniem optymalnych stron WWW przy wykorzystaniu języków HTML, XHTML i CSS.

Szkolenie tworzenie stron www skierowane jest dla wszystkich, którzy w swojej pracy spotykają się z zarządzaniem lub tworzeniem serwisów www lub chcieli by samodzielnie rozpocząć swoją przygodę w świecie webmasterów.

Szkolenie JavaScript

JavaScript jest to skryptowy język programowania wysokiego poziomu. Razem z HTML i CSS stanowi podstawowe narzędzie do tworzenia stron internetowych, jednak posiada również szersze zastosowania - jest używany w aplikacjach desktopowych i dokumentach PDF oraz w aplikacjach webowych po stronie serwera.

Podczas szkolenia poznasz skladnię jezyka JavaScript, zmienne i ich zasięgi, typy danych, operatory logiczne i arytmetyczne. Dowiesz się czym są instrukcje warunkowe, pętle i jak i używać. Szkolenie przekaże wiedzę z zakresu Document Object Model, Hoisting, Clousers, pogramowania funkcyjnego oraz podstaw programowania obiektowego. Dowiesz się również czym jest asynchroniczne programowanie i jak go używać.

Szkolenie jest oparte o standard ECMAScript2015(ES6).

Program

  1. Wprowadzenie
    • czym jest JavaScript i gdzie sie go używa
    • JavaScript i przeglądarka
  2. Podstawy JavaScript
    • zmienne i typy danych
    • operatory logiczne i arytmetyczne
    • instrukcje warunkowe
    • pętle
    • zasięgi zmiennych
    • hoisting
  3. Document Object Model
    • co to jest DOM
    • pobieranie i dodawanie elementu do DOM
    • modyfikowanie elementu DOM
    • nasłuchiwanie zdarzeń
  4. Funkcje
    • wprowadzenie do funkcji
    • funkcje strzałkowe
    • wyrażenia funkcyjne vs deklaracje funkcji
    • czy jest this oraz bind/call/apply
    • spread/rest operator
  5. Asynchroniczne programowanie
    • JSON
    • protokól HTTP
    • callback function - co to jest?
    • promise object - podstawy
    • async/await - podstawy
  6. Zamknięcia - clousers oraz zasięg leksykalny
  7. Podstawy programowania obiektowego
    • prototypy
    • class i jak ja stworzyć
    • konstruktory - podstawy
    • dziedziczenie - podstawy

Szkolenie WordPress

WordPress jest prostym i bardzo uniwersalnym systemem zarządzania treścią stron www. Pomimo tego rozbudowany system wtyczek i modułów pozwala wykorzystać go także w bardziej zaawansowanych projektach. WordPress to obecnie najpopularniejszy system typu CMS (Content Management System).

Szkolenie WordPress pozwoli Ci opanować obsługę tego narzędzia jako systemu CMS. Skierowane jest do wszystkich, którzy chcą zajmować się tworzeniem stron w Internecie, zarówno użytkowników prywatnych chcących rozpocząć własną przygodę z blogowaniem jak i pracowników publikujących codziennie nowe wpisy na stronie firmy. Nauczymy Cię konfiguracji i obsługi Wordpressa, obsługi wtyczek i modułów umożliwiających użytkownikom zmianę wyglądu i funkcjonalności, a także zarządzania treścią.

Program

  1. Wprowadzenie
    • co to jest WordPress
    • zasady działania systemów CMS
    • specyfika WordPressa
  2. Zarządzanie instalacją WordPress
    • instalacja WordPressa, podstawowa konfiguracja
    • działania administracyjne
    • kwestie bezpieczeństwa
    • odzyskiwanie hasła
  3. Wypełnianie strony treścią
    • ustawienia
    • korzystanie z toolbaru
    • zasoby
    • moduły
  4. Organizacja pracy i zarządzanie użytkownikami
    • użytkownicy i grupy
    • poziomy dostępu, uprawnienia
    • dziedziczenie uprawnień, ACL
  5. Zamieszczanie reklam i banerów
  6. Korzystanie z szablonów, modułów i dodatków, instalacja pluginów
  7. Zagadnienia zaawansowane
    • aktualizacja WordPress
    • zabezpieczenie przed spamem
    • pisanie i dystrybucja własnych szablonów
    • zaawansowane zarządzanie wyglądem witryny
